About Praxis

Interactive AI tutoring for hands-on programming practice • Built with Vite, React, and TypeScript

Praxis is an in-browser, AI-driven programming tutor. The goal is to help learners build skills by solving small, incremental coding challenges while receiving supportive, Socratic guidance from an AI tutor.

What you can do

  • Describe a programming task or ask for a learning exercise via the chat panel.
  • Receive a short explanation and a focused coding challenge from the AI (the model also provides a small set of automated tests).
  • Edit and run code in the embedded Monaco editor, then run the provided tests to validate your solution.
  • View console output and per-test results, get tailored hints, and progress to more advanced challenges as you succeed.

How it works (technical overview)

  1. The app sends the conversation + a strict system prompt to an LLM (Gemini) through @google/genai. The LLM returns a JSON object with an explanation, a challenge, and an array of tests.
  2. Tests are small JavaScript boolean expressions (for example, add(2,3) === 5). They are evaluated client-side in a sandboxed Web Worker to keep the UI responsive and isolated from execution.
  3. The system prompt enforces pedagogical rules: don't provide final code solutions, use a Socratic tone, and always return a single JSON response matching the expected schema.

Privacy & security

  • The app requires you to provide an API key for the AI service. The key is used at runtime and is not persisted by the client (it is stored only in memory during the session).
  • All code execution is local to your browser inside a Web Worker. However, executing arbitrary code in-browser has inherent risks; use discretion when running untrusted snippets.
  • No user data is sent to the repository owners by default — only interactions routed through the AI provider go to the API according to that provider's terms.

Local development

To run Praxis locally:

git clone https://github.com/your-username/praxis.git
cd praxis
npm install
npm run dev

Customize the tutor

If you want to change the tutor's behavior (tone, rules, curriculum), edit src/constants.ts. That file contains the system prompt which defines the tutor persona and interaction rules. If you change the response schema, update src/App.tsx to parse the new format.
